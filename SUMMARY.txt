PROJECT: PixelPerfect AI - Design Intent Detection Landing Page
================================================================================

OVERVIEW:

design intent detection tool. The platform helps designers and developers
understand design vision and deliver pixel-perfect results through intelligent
automation.

TECHNICAL STACK:
================================================================================

Frontend Framework: Next.js 16 (App Router)
UI Library: React 19
Styling: Tailwind CSS v4
Animations: Framer Motion
Language: TypeScript
Font: Geist (sans-serif, monospace)
Color System: Dark theme with purple/blue/pink gradients

================================================================================
DESIGN SYSTEM:
================================================================================

COLOR PALETTE (5 colors):
- Background: Dark Charcoal (oklch 0.08 0 0)
- Card/Glass: Charcoal with transparency (oklch 0.12 0 0)
- Primary Gradient: Purple to Pink (oklch 0.7 0.25 290°)
- Secondary: Blue accent (oklch 0.5 0.2 250°)
- Accent: Purple glow (oklch 0.65 0.22 300°)

TYPOGRAPHY:
- Headings: Geist Sans (font-weight: 700-800)
- Body: Geist Sans (font-weight: 400-500)
- Code: Geist Mono

EFFECTS:
- Glassmorphism: backdrop-blur-xl with white/5 background
- Glowing Gradients: Multi-color gradient shadows
- Border Effects: White/10 transparent borders
- Animations: Smooth fade-ins, hover effects, scroll triggers

================================================================================
CUSTOMIZATION GUIDE:
================================================================================

To customize the landing page:

1. UPDATE COLORS
   - Edit design tokens in app/globals.css
   - Modify oklch color values in :root and .dark selectors
   - Update gradient colors in utility classes

2. MODIFY CONTENT
   - Edit text in each section component (components/sections/)
   - Update feature descriptions, pricing, testimonials, etc.
   - Customize company name and product name throughout

3. CHANGE FONTS
   - Update font import in app/layout.tsx
   - Modify font-sans, font-mono in globals.css
   - Apply font classes to components

4. ADD IMAGES
   - Place images in public/ folder
   - Update img src attributes in components
   - Use next/image for optimization

5. ADJUST ANIMATIONS
   - Modify Framer Motion variants in components
   - Update duration, delay, and easing properties
   - Add scroll triggers for new animation effects

6. CUSTOMIZE SECTIONS
   - Edit section components individually
   - Modify grid layouts, card counts, content
   - Add or remove sections as needed
