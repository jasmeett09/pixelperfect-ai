# 🚀 PixelPerfect AI

PixelPerfect AI is a Next.js-based dashboard that compares Figma designs with real UI implementations and highlights visual differences using AI.

---

## 🌟 Features

- 🎯 Compare Figma design vs live UI
- 📐 Detect spacing, alignment, and layout issues
- 🔤 Typography mismatch detection
- 🎨 Color inconsistency detection
- 🤖 AI-powered insights for UI improvements
- 📊 Clean dashboard for developers & designers

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React
- **Styling:** Tailwind CSS
- **API:** Figma API
- **Deployment:** Vercel

---

## ⚙️ Setup Locally

```bash
git clone https://github.com/YOUR_USERNAME/pixelperfect-ai.git
cd pixelperfect-ai
pnpm install
pnpm dev
```
---

🔐 Environment Variables

Create a .env.local file and add:

FIGMA_ACCESS_TOKEN=your_token_here

⚠️ Do not commit this file.
